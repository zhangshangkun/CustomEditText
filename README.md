# CustomEditText
自定义一个密码输入框


Add it in your root build.gradle at the end of repositories:
	allprojects {
		repositories {
			...
			maven { url 'https://www.jitpack.io' }
		}
	}
	
	
Step 2. Add the dependency
	dependencies {
	        implementation 'com.github.zhangshangkun:CustomEditText:Tag'
	}
